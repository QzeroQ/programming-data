# 栈

---
## 简介

栈也是⼀种线性结构，相⽐数组，栈对应的操作是数组的⼦集，只能从⼀端添加元素，也只能从⼀端取出元素，这⼀端称为栈顶，栈是⼀种后进先出的数据结构，`Last In First Out (LIFO)`，在计算机的世界⾥，栈拥有着不可思议的作⽤。

---
## 栈的应⽤

- ⽆处不在的 Undo 操作（撤销）。
- 程序调⽤的系统栈。

---
## 栈的实现

抽象栈的基础操作

```java
public interface Stack<E> {
    void push(E e);
    E pop();
    E peek();
    boolean isEmpty();
    int size();
}
```

可以使用数组或者链表实现栈
